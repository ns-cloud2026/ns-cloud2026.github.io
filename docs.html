NS CLOUD กลับหน้าแรก

คู่มือและเอกสารการใช้งาน

ทำความเข้าใจขั้นตอนการใช้งานเซิร์ฟเวอร์ GPU, นโยบายความเป็นส่วนตัว และข้อตกลงเกี่ยวกับการควบคุมระบบ

01

การเลือกใช้งานเครื่องมือ (AI Tools)

ในการติดต่อเช่าเซิร์ฟเวอร์ GPU คุณสามารถระบุเครื่องมือ (Tools/WebUI) ที่ต้องการใช้งานในการทำงานได้ตามความต้องการ เช่น ComfyUI, ConfigUI, Ollama, PyTorch, Stable Diffusion, vLLM เป็นต้น

ทาง Admin จะทำการติดตั้งสภาพแวดล้อมการทำงานและเครื่องมือที่คุณเลือกให้พร้อมใช้ และจัดส่ง Link URL ตรงสำหรับเข้าใช้งานผ่านหน้าเว็บเบราว์เซอร์ให้คุณทันทีหลังจากเริ่มบริการ

02

การควบคุมระบบ (Linux OS & SSH Access)

เซิร์ฟเวอร์ของ NS CLOUD ทำงานบนระบบปฏิบัติการ Linux เป็นหลัก (ไม่มีระบบ GUI หน้าจอ Desktop ของ Windows หรือ Linux Desktop ทั่วไป) เพื่อประสิทธิภาพการประมวลผลสูงสุดและตัดทรัพยากรส่วนเกินออก

หากคุณมีความจำเป็นต้องการเข้าควบคุม จัดการไฟล์ รันสคริปต์ หรือตั้งค่าตัวเครื่องด้วยตนเองเพิ่มเติม คุณสามารถแจ้งขอรับสิทธิ์เข้าใช้งานผ่านทาง SSH Credentials (IP, Port, Username, Password) จาก Admin ได้โดยตรง

ssh username@server_ip -p port SSH Terminal
03

นโยบายความเป็นส่วนตัว (User Privacy)

NS CLOUD ยึดมั่นในเรื่องความเป็นส่วนตัวและความปลอดภัยของข้อมูลลูกค้าเป็นสำคัญที่สุด:

✓ ทางเราจะไม่มีการเข้าไปยุ่งเกี่ยว แทรกแซง รันคำสั่ง หรือเข้าถึงข้อมูลใดๆ ในเซิร์ฟเวอร์ระหว่างการเช่าใช้งานของคุณเด็ดขาด

พื้นที่ประมวลผลและไฟล์งานทั้งหมดจะถูกแยกออกจากกันอย่างเด็ดขาด (Isolated Environment) เพื่อรับประกันความปลอดภัยของโปรเจกต์ของคุณ

04

การต่อเวลาเช่าเซิร์ฟเวอร์ (Extension Policy)

คุณสามารถขอเพิ่มระยะเวลาเช่าเครื่องเพิ่มได้ตามตารางการใช้งาน โดยมีเงื่อนไขดังนี้:

⚠️
สำคัญมาก: หากต้องการต่ออายุการใช้งานเซิร์ฟเวอร์เพื่อความต่อเนื่อง กรุณาติดต่อขอต่อเวลากับ Admin ล่วงหน้าอย่างน้อย 20 นาทีก่อนหมดชั่วโมงเช่าเดิม เพื่อสิทธิในการต่อเวลาก่อนเครื่องจะคิวเต็มหรือถูกตัดระบบ
05

นโยบายเมื่อหมดเวลาเช่า (Expiration Policy)

เพื่อให้ระบบคิวการจัดสรร GPU ทำงานได้เต็มประสิทธิภาพและเป็นธรรมแก่ผู้ใช้อื่นๆ:

🚨
แจ้งเตือนการหยุดการทำงาน: เมื่อสิ้นสุดระยะเวลาเช่าที่ได้ตกลงกันไว้และไม่มีการแจ้งต่อเวลาก่อนกำหนด เซิร์ฟเวอร์จะทำการตัดการทำงานและปิดเครื่องทันที

โปรดทำการบันทึกความคืบหน้าของโมเดล สำรองข้อมูล (Backup) หรือโอนย้ายไฟล์งานสำคัญออกให้เสร็จสิ้นก่อนถึงเวลาปิดเครื่อง เพื่อป้องกันการสูญหายของข้อมูล
